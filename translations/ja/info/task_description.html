あなたの装備と宝物が詰まった背嚢は実に重くなりましたがこれは船の損失を補うだけの金銭と冒険の成功による祝福によるもので歓迎すべき負担でした。 もちろん、あなたの一番の望みは家に帰ることです。 しかし、あなたの本能は目的を遂げられることを告げていました。

金庫のある部屋を出た後、あなたは地下牢に向かいました。 そこで見たものによりあなたの楽観は少々失われました。 二つの骸骨が地下牢の入り口すぐ後ろに横たわっていたのです。 その様子から彼らがトレジャーハンターであったことが分かります。 骨に張り付いた色あせた衣類から察するに多くの年月が経過しているようです。 しかしなぜ彼らは死んだのでしょうか？

「カチ...」 かすかな音がして地下牢のドアが閉まりました。 あなたはすぐに駆け出しましたが遅すぎました。 無論、鍵穴は無くドアを開けるための手掛かりも他にありません。 あなたはドアを蹴り飛ばしましたが頑丈過ぎました。 もし餓死して二体の骸骨の仲間入りをしたくなければ脱出する方法を見つける他ありません。 おそらく骸骨たちのそばに何か使えるものが見つかるでしょう。 近づいて詳しく調べてみるべきです。

骸骨に近づくと何処からともなく声が聞こえてきました。 「...ゲームを...終わらせよ...」 三つほど考えられます: だれかこの城に生存者がいる、 幽霊が現れた、 あなたが正気を失いつつありその初期症状としての幻聴。 どれも考えたくないものです。 しかし引き返す方法がありません。 ゲームを終わらせてハイパーキュープの探索を続けるのです。

それは二人のプレイヤーが山から交互に 1 から 3 の任意の数の石を取っていくあなたが良く知っているゲームに似ています。 この問題を少し一般化しましょう。 両プレイヤーが 1 から 3 の任意の数ではなく、k1, k2 ... km 個の石を取れることにします。 我々の関心は両プレイヤーが最善を尽くした場合にどちらが常に勝利するかです。
ゲームの開始時に山にある石の数と手番に取ることのできる石の数が入力値です。 例 - (10, [1, 2, 3])
もし山にある石の数が自分の番で取れる最小の石の数と同じか少ない場合にそのプレイヤーは敗北します。

入力: 山にある石の総数と手番に取ることのできる石の数

出力: 勝者 (1 か 2)
